## 题目描述
已经是2024年了，题目的创意早已枯竭，奥林匹克现在以修改后的个人形式进行。奥林匹克由$n$道题目组成，编号从$1$到$n$。第i道题目有自己的分数$a_i$和一个特定的参数$b_i(1 ≤ b_i ≤ n)$。

最初，测试系统给参与者提供第一道题目。当参与者获得第i道题目时，他们有两个选择：
- 他们可以提交该题目并获得$a_i$分；
- 他们可以跳过该题目，这样他们将永远无法提交它。

然后，测试系统从索引为$j$的题目中为参与者选择下一道题目，满足以下条件：
- 如果他提交了第 $i$ 道题目，它会查看索引为$j < i$的题目；
- 如果他跳过了第 $i$ 道题目，它会查看索引为$j ≤ b_i$的题目。

在这些题目中，它选择一个最大索引的题目，该题目是它之前未给过参与者的（他之前既没有提交也没有跳过）。如果没有这样的题目，那么参与者的比赛结束，他们的结果等于所有提交题目的分数之和。特别地，如果参与者提交了第一道题目，那么他们的比赛就结束了。请注意，参与者每道题目最多只会获得一次。

普霍尔为奥林匹克做了充分的准备，现在他可以提交任何问题。帮助他确定他可以获得的最大分数。

## 输入格式
第一行输入一个整数$n(1 ≤ n ≤ 2000)$。
接下来$n$行，每行输入两个整数$a_i$和$b_i(1 ≤ a_i ≤ 10^9, 1 ≤ b_i ≤ n)$。

## 输出格式
输出普霍尔可以获得的最大分数。

## 数据范围
- $1 ≤ n ≤ 2000$
- $1 ≤ a_i ≤ 10^9$
- $1 ≤ b_i ≤ n$
- 输入中的所有值都是整数。
