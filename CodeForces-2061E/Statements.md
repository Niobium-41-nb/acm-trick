## 题目描述
Kevin 有一个长度为 $n$ 的整数序列 $a$。同时，他拥有 $m$ 种魔法，第 $i$ 种魔法可以用一个整数 $b_i$ 来表示。
Kevin最多可以施展 $k$ 次魔法操作（也可以不施展）。每次操作，他可以：
- 选两个下标 $i$ $(1 ≤ i ≤ n)$ 和 $j$ $(1 ≤ j ≤ m)$，然后将 $a_i$ 更新为 $a_i$ & $b_j$。这里，& 表示按位与运算。
请你帮 Kevin 找出，在最多施展 $k$ 次操作后，序列 $a$ 中所有数字的和能达到的最小值。

## 输入格式
第一行输入三个整数 $n$, $m$, $k$ $(1 ≤ n, m ≤ 100, 0 ≤ k ≤ 100)$。
第二行输入 $n$ 个整数 $a_1$, $a_2$, ..., $a_n$ $(0 ≤ a_i ≤ 10^9)$。
第三行输入 $m$ 个整数 $b_1$, $b_2$, ..., $b_m$ $(0 ≤ b_j ≤ 10^9)$。

## 输出格式
输出一个整数，表示在最多施展 $k$ 次操作后，序列 $a$ 中所有数字的和能达到的最小值。

## 数据范围
- $1 ≤ n, m ≤ 100$
- $0 ≤ k ≤ 100$
- $0 ≤ a_i, b_j ≤ 10^9$
- 输入中的所有值都是整数。
