## 题目描述

一个正常的括号序列是指可以通过在序列的原始字符之间插入字符"1"和"+"来转换为正确的算术表达式的括号序列。例如：

- 括号序列"()"和"(())"是正常的（生成的表达式分别为："(1)+(1)"和"((1+1)+1)"）；
- 括号序列")(", "("和")"不是。

给定一个正常的括号序列。在一次移动中，你可以移除一对相邻的括号，其中左括号是开放括号，右括号是闭合括号。然后连接结果部分而不改变顺序。这个移动的成本是这对括号右侧的括号数量。

正常括号序列的成本是使序列变为空所需的最小总成本。

实际上，你并没有移除任何括号。相反，你得到了一个正常的括号序列和一个整数 $k$ 。你可以执行以下操作最多 $k$ 次：

- 从序列中提取一些括号，并将其插回任何位置（在任意两个括号之间，在开头或结尾；可能在之前的相同位置）。

执行所有操作后，括号序列必须是正常的。在进行最多 $k$ 次操作后，结果正常括号序列的最小可能成本是多少？

## 输入格式

第一行输入一个整数 $t(1 ≤ t ≤ 10^4)$ ，表示测试用例的数量。
每个测试用例的第一行包含一个正常的括号序列 $s(1 ≤ |s| ≤ 10^5)$。
第二行输入一个整数 $k(0 ≤ k ≤ 10^5)$。
所有测试用例的 $s$ 长度总和不超过 $10^5$ 。

## 输出格式

对于每个测试用例，输出一个整数，表示执行最多 $k$ 次操作后，结果正常括号序列的最小可能成本。

## 数据范围

- $1 ≤ t ≤ 10^4$
- $1 ≤ |s| ≤ 10^5$，所有测试用例的$s$长度总和不超过$10^5$
- $0 ≤ k ≤ 10^5$
- 输入中的所有值都是整数。
