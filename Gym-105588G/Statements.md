# Gym-105588G

## 题目描述
为什么这么多出题人觉得把GCD题目放在G题位置很有趣？有人能理解吗？
——红烧鸡块

给定两个整数$a$和$b$，每轮你可以执行以下两种操作之一：
- 若$a > 0$，则将a的值减少$gcd(a, b)$
- 若$b > 0$，则将b的值减少$gcd(a, b)$

Grace想知道最少需要多少轮操作才能使a和b都变为0。

$gcd(x, y)$表示$x$和$y$的最大公约数。例如$gcd(6, 8) = 2, gcd(7, 5) = 1。gcd(x, 0)$和$gcd(0, x)$的值定义为$x$。

## 输入格式
一行两个整数$a$和$b$。

## 输出格式
输出最少需要的操作轮数。

## 数据范围
（题目未明确给出，根据类似题目推测合理范围）
- $0 ≤ a, b ≤ 10^{18}$
- 输入中的所有值都是整数。
